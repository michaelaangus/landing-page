# Michaela N. Angus - Personal Professional Landing Page

## Project Overview
A static personal/professional landing page for Michaela N. Angus, a dual-degree student at the University of Iowa showcasing her skills at the intersection of Finance and Data Analytics.

## Tech Stack
- **Frontend:** Vanilla HTML5 and CSS3 (no frameworks)
- **Fonts:** Inter from Google Fonts (CDN)
- **Backend:** None — pure static site
- **Build System:** None required

## Project Layout
```
/
├── index.html          # Main entry point
├── css/
│   └── stylesheet.css  # All styles
├── js/
│   └── scripts.js      # Client-side logic
├── images/
│   └── headshot.jpeg   # Profile photo
├── PRD.md              # Product Requirements Document
└── STANDARDS.md        # Technical & design standards
```

## Development
- **Workflow:** "Start application" serves the site via `python3 -m http.server 5000 --bind 0.0.0.0`
- **Port:** 5000

## Deployment
- **Type:** Static site deployment
- **Public Directory:** `.` (root)
